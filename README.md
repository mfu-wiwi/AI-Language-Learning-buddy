# AI Language Learning Buddy

University mini Group Project with n8n + Web Chat UI

## Description

An AI-powered language learning assistant that combines the power of n8n workflow automation with an interactive web chat interface. This project aims to provide personalized language learning experiences through conversational AI.

## Features

- Interactive web-based chat interface
- AI-powered language learning assistance
- Workflow automation using n8n
- Personalized learning paths
